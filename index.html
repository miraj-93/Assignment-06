Frequently Asked Question

What Is the difference between var, let, and const?
1. "var" is Function-scoped,Hoisted , Reassignable & Redeclarable.
2. "Let"is Block Scoped, Hoisted but not initialized, Reassignable but Not Redeclarable.
3. "const" is Block-scoped, Hoisted but not initialized, Cannot be reassigned or redeclared, Must be initialized when declared.
What Is the difference between map(),forEach() and filter()?
1. map() - Returns a New Array.Use map() when you need a new transformed array.
2. forEach() - Just Loops, No ReturnUse.Use forEach() when you just want to execute code for each item (no need for a return)..
3. filter() - Returns a New Array with Condition.Use filter() when you want to extract elements based on a condition.
What Are Arroe Function,and how are they different from regular function?
Arrow functions have a shorter syntax, inherit this from the surrounding scope and lack the arguments object where as regular functions have their own this support arguments and can be hoisted.
How do Javascripts Promises work?
A Promise in JavaScript is an asynchronous operation that will either resolve (success) or reject (failure) in the future. It helps in handling asynchronous tasks like fetching data, reading files, or API calls without blocking the execution.
How does Closure work in JavaScripts?
Closures allow functions to access variables from their parent function’s scope, even after the parent function has returned.
